Windows不重启刷新环境变量
发表于2022-08-08|Windows
Windows不重启刷新环境变量的方法如下： 修改环境变量 打开cmd，执行 echo %path% 查看当前系统的环境变量，为修改前的值 执行 set path=x，x 可以为任意值，无实际意义，此时查看path会显示为x 重启cmd，执行 echo %path%，显示为修改后的环境变量，刷新成功 原因：DOS窗口的环境变量为Windows环境变量的一个副本，对副本的修改不会改变Windows环境变量，但会引起Windows环境变量的刷新。
